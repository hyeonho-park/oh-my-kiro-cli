# 도메인 분류 체계

## 도메인

| 도메인 | 설명 | 디렉토리 |
|--------|------|----------|
| troubleshooting | 문제 해결 가이드 및 지원 티켓 해결 기록 | wiki/troubleshooting/ |
| infrastructure | 인프라 구성, 설정, 운영 가이드 | wiki/infrastructure/ |
| processes | 팀 프로세스, 워크플로우, 정책 | wiki/processes/ |
| architecture | 아키텍처 결정, 설계 문서 | wiki/architecture/ |
| onboarding | 온보딩 자료, 신규 입사자 가이드 | wiki/onboarding/ |

## 태그 어휘

### 심각도
- P0: 서비스 전면 장애
- P1: 주요 기능 장애
- P2: 부분 기능 장애
- P3: 경미한 이슈

### 상태
- current: 최신 상태
- stale: 소스 변경 후 위키 미갱신
- conflict: 소스 간 모순 발견
- retired: 폐기됨

### 관계 유형
- caused-by: A가 B의 원인
- resolved-by: A가 B로 해결됨
- related-to: 관련 문서
- supersedes: A가 B를 대체
